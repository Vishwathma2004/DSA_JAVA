# DSA_JAVA

This repository contains various solutions for LeetCode problems and some of the classic Data Structure and Algorithm (DSA) related codes implemented in Java.

## Repository Contents

The repository includes:

- **Sorting Algorithms**: Bubble Sort, Cyclic Sort, Selection Sort, etc.
- **Searching Algorithms**: Binary Search, Order-Agnostic Binary Search, Rotated Binary Search, etc.
- **LeetCode Solutions**: Solutions to various problems like Missing Number, Find Peak in Mountain, Max Wealth, etc.
- **Pattern Programs**: Various pattern-based problems solved using recursion and iteration.
- **Matrix and Array Problems**: Problems like Search in 2D Array, Sorted Matrix, Row and Column Search, etc.
- **Miscellaneous**: Other useful programs such as Pretty Printing, Palindrome checks, etc.

## Folder Structure

- **src**: Contains the main Java files for different algorithms and problem solutions.
- **.gitignore**: Git ignore file to exclude unnecessary files.
- **README.md**: This file, explaining the repository.
  
## How to Use

1. Clone the repository to your local machine:
    ```bash
    git clone https://github.com/Vishwathma2004/DSA_JAVA.git
    ```

2. Navigate to the project directory:
    ```bash
    cd DSA_JAVA
    ```

3. Open any Java file in your preferred IDE or text editor.

4. To run a specific program:
    ```bash
    javac <ProgramName>.java
    java <ProgramName>
    ```
